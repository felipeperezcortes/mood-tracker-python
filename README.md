# Mood Tracker (CLI en Python)

Aplicación de línea de comandos desarrollada en Python para registrar y evaluar factores diarios relacionados con el bienestar mediante un flujo interactivo por consola.

El programa solicita información como estado de ánimo, sueño, estrés, alimentación, actividad física y otros hábitos, asigna un puntaje a cada respuesta y entrega una evaluación general del día junto con recomendaciones simples.

## Objetivo del proyecto
Practicar fundamentos de Python en un contexto aplicado, trabajando principalmente con:
- Control de flujo
- Validación de entradas de usuario
- Lógica condicional
- Uso de listas y cálculos simples

Este proyecto también apoya mi formación en **QA**, reforzando el pensamiento estructurado, la validación de escenarios y el manejo de flujos lógicos.
